# Teste Santander Android

Aplicativo Android de teste.

## Pré requisitos

- Android Studio 3.1.1
- Gson ('com.google.code.gson:gson:2.8.5')

## Para Começar

- Executar Android Studio e carregar o projeto;
- Run Application.

## Linguagem de desenvolvimento

- Java

## Utilizado

- OkHttpClient
- ConstraintLayout
- Fragments
- JUnit4
